# MathTree OCR React + Tailwind

Aplicacion PWA creada con React, Vite y Tailwind CSS.

## Funciones

- Interfaz tipo escaner profesional.
- Camara del dispositivo.
- Subida de imagen desde archivo.
- OCR con Tesseract.js.
- Limpieza automatica de formula.
- Parser con prioridad de operadores y parentesis.
- Arbol binario dibujado en Canvas.
- Animacion de resolucion recursiva.
- Lista para GitHub y Vercel.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Publicar en Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Nota

Esta entrega es una PWA instalable desde navegador. En Android se puede usar la opcion "Instalar app" o "Agregar a pantalla de inicio".
